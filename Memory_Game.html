Instructions

Memory, aka. Match Match, Shinkei-suijaku, etc. is a game with pairs of images. The images will be laid out face down then the object of the game is to overturn as many pairs as possible whilst only turning two cards per go, alternating turns between players. Only when a pair is found the player gets another go. The person who matched most pairs is the winner.
